### **1. 천간의 합(天干의 合)**
#천간합

천간은 아래와 같은 다섯 쌍의 합 관계를 가집니다. 
천간의 합은 음양 조화를 이루며, 특정 성질을 나타냅니다.

| 천간 1     | 천간 2     | 속성   |
| -------- | -------- | ---- |
| **甲(갑)** | **己(기)** | 土(토) |
| **乙(을)** | **庚(경)** | 金(금) |
| **丙(병)** | **辛(신)** | 水(수) |
| **丁(정)** | **壬(임)** | 木(목) |
| **戊(무)** | **癸(계)** | 火(화) |

---

### **2. 천간의 충(天干의 沖)**
#천간충

천간의 충은 서로 상극 관계를 나타내며, 아래와 같은 다섯 쌍이 있습니다.

| 천간 1     | 천간 2     | 설명        |
| -------- | -------- | --------- |
| **甲(갑)** | **庚(경)** | 木과 金의 대립  |
| **乙(을)** | **辛(신)** | 木과 金의 대립  |
| **丙(병)** | **壬(임)** | 火와 水의 대립  |
| **丁(정)** | **癸(계)** | 火와 水의 대립  |
| **戊(무)** | **己(기)** | 土의 동일성 대립 |

---

### **3. 지지의 합(地支의 合)**
#지지합
지지는 삼합과 육합의 두 가지 합 관계를 가집니다.

#### **① 지지의 삼합(三合)**
#지지삼합

삼합은 같은 오행(五行) 속성으로 이루어진 세 개의 지지가 강한 조화를 이루는 관계입니다.

|그룹|지지 조합|오행|
|---|---|---|
|**해(亥)**, **묘(卯)**, **미(未)**|亥-卯-未|木|
|**인(寅)**, **오(午)**, **술(戌)**|寅-午-戌|火|
|**사(巳)**, **유(酉)**, **축(丑)**|巳-酉-丑|金|
|**신(申)**, **자(子)**, **진(辰)**|申-子-辰|水|

#### **② 지지의 육합(六合)**
#지지육합

육합은 두 개의 지지가 짝을 이루어 조화를 이루는 관계입니다.

| 지지 1     | 지지 2     | 합의 결과 (속성) |
| -------- | -------- | ---------- |
| **자(子)** | **축(丑)** | 土(토)       |
| **인(寅)** | **해(亥)** | 木(목)       |
| **묘(卯)** | **술(戌)** | 火(화)       |
| **진(辰)** | **유(酉)** | 金(금)       |
| **사(巳)** | **신(申)** | 水(수)       |
| **오(午)** | **미(未)** | 土(토)       |

---

### **4. 지지의 충(地支의 沖)**
#지지충

지지의 충은 서로 상극 관계를 나타냅니다.

|지지 1|지지 2|설명|
|---|---|---|
|**자(子)**|**오(午)**|水와 火의 대립|
|**축(丑)**|**미(未)**|土의 대립|
|**인(寅)**|**신(申)**|木과 金의 대립|
|**묘(卯)**|**유(酉)**|木과 金의 대립|
|**진(辰)**|**술(戌)**|土의 대립|
|**사(巳)**|**해(亥)**|火와 水의 대립|

---

#### **③ 방합(方合)**

- 방합은 네 개의 지지가 방향(방위)에 따라 조화를 이루는 관계입니다.
- 방합은 동서남북의 방위를 기준으로 나뉘며, 하나의 오행으로 통일됩니다.
#지지방합

|방위|지지 조합|오행|
|---|---|---|
|**동방(東方)**|**인(寅)** - **묘(卯)** - **진(辰)**|木|
|**남방(南方)**|**사(巳)** - **오(午)** - **미(未)**|火|
|**서방(西方)**|**신(申)** - **유(酉)** - **술(戌)**|金|
|**북방(北方)**|**해(亥)** - **자(子)** - **축(丑)**|水|
### **정리**

- **천간의 합(合)**: 음양 조화와 특정 속성 생성.
- **천간의 충(沖)**: 상극 관계.
- **지지의 삼합(三合)**: 같은 오행으로 이루어진 세 개의 지지가 조화.
- **지지의 육합(六合)**: 두 개의 지지가 짝을 이뤄 조화.
- **지지의 충(沖)**: 상극 관계로 갈등을 나타냄.
